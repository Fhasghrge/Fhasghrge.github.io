类型变量

这里的 小写的 a 就是类型变量type variable ， 和TypeScript中的泛型很像

使用类型变量可以写出多态函数polymorphic function

类型类

类型类是定义行为的接口，但是对行为的类型没有约束；换句话说，类型类 是很多类型的同一行为的集合

如果一个类型是某个类型类的实例，那么它必然实现了该类型类所定义的行为

如果一个行为的对象没有指定类型，那么这个行为就是 类型类（不具有类型约束的函数？）

类型类的实例就是针对某个特定类型的行为

这类通用的函数有：

  • Eq: ==

  • Ord > < >= <=

  • Show Any -> String

  • Read String -> Any(需要显式或者隐告诉Read函数返回的值)

    • 显式： 类型注解(type annotation)

    • 隐式注解： 通过传入的第一个参数决定返回值的类型

  • Enum类

  • Bounded类 （successer, predecesor)

  • Num类（注意这和Int， Interger，Float，Double不同），每个数字都是一个行为，而它被指定任意的类型
