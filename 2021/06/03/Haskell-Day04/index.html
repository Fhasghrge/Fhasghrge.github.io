高阶函数

柯里函数

HS中所有的函数都只有一个参数；
HS函数在每次调用的时候只取一个参数，并且返回一个一元函数取下一个参数

好处是： 只传入部分参数就可以构造一个部分应用函数（partial application / specialize)

中缀函数如何实现柯里化？

截断 section，
将中缀函数的一侧传入参数，然后用括号包裹起来，就形成了一元函数（实际上HS中函数都是一元函数）

如果函数需要一元函数作为参数，可以通过部分应用（partial application）是函数只剩下一个参数，例如在js中map、filter等函数

Currying and partially applied functions enable higher order functions to be used much more effectively and concisely.
